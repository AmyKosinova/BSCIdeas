Compilation
Program is compiled by ant using: ant -f <path to folder containing build.xml> -Dnb.internal.action.name=rebuild clean jar

Running
program can be started by command: java -jar BSCIdeas_intro.jar from build destination directory.
  • optional parameters:
    • payments file as first parameter in format of absolute path or relative to run space
  • supported commands
    • add payment formated as CURRENCY AMMOUNT. Ex: EUR 344
      Note: currently supported list of currencies: AUD, CAD, CHF, CZK, DKK, EUR, GBP, HKD, HUF, JPY, NOK,NZD, PLN, SEK, SGD, USD, ZAR, RMB
    • load payments from file formated as $load <path to file> in case of loading multiple files, all will be imported. In case of corrupted file, no data will be imported.
    • quit to exit application
